# Arrivals to Thailand
## รายละเอียดโปรเจค
วิเคราะห์ว่าในปี 2016-2018 ชาวต่างชาติเข้ามาในประเทศเพิ่มหรือลดเท่าไรในแต่ละปี ฤดูที่มีชาวต่างชาติเข้ามาในประเทศมากที่สุด ทวีปและประเทศที่มีคนเข้ามาในประเทศไทยมากที่สุด
