## รายละเอียดโปรเจค
วิเคราะห์ว่าในปี 2016-2018 ในแต่ละปีชาวต่างชาติเข้ามาในประเทศเพิ่มหรือลดเท่าไร ในแต่ละฤดูมีชาวต่างชาติเข้ามาในประเทศเท่าไหร่ 10อันดับทวีปและประเทศที่มีคนเข้ามาในประเทศไทยมากที่สุด
